# Customer Churn Prediction

A full end-to-end Machine Learning pipeline built on the Telco Customer Churn dataset.

## Project Overview
The goal is to predict which customers are likely to cancel their subscription,
allowing a business to intervene before they leave.

## Dataset
- 7,043 customers, 20 features
- Target variable: Churn (Yes / No)
- 26.5% churn rate

## What's Inside
- Full Exploratory Data Analysis (EDA)
- Data cleaning and preprocessing
- Training and comparing 4 ML models
- Model evaluation using ROC-AUC, Precision, Recall and F1-Score
- Feature importance and business recommendations

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting

## Tools
Python, Pandas, Scikit-learn, Matplotlib, Seaborn

## How to Run
1. Clone the repo
2. Open customer_churn_pipeline.ipynb in Jupyter or Google Colab
3. Upload the CSV file to the same directory
4. Run all cells
